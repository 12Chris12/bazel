join_paths

string cmd_helper.join_paths(separator, files)

Creates a single command line argument joining the paths of a set of files on the separator string.

Parameters

Parameter Description
separator

string

the separator string to join on
files

set of Files

the files to concatenate

template

list cmd_helper.template(items, template)

Transforms a set of files to a list of strings using the template string.

Parameters

Parameter Description
items

set of Files

The set of structs to transform.
template

string

The template to use for the transformation, %{path} and %{short_path} being substituted with the corresponding fields of each file.
